## Como arrancar el proyecto

1. Clonar el proyecto
2. Colocarse en la ruta del proyecto y escribir en la misma consola ``` npm i ``` 
3. Ejecutar ```npm run start:dev ```

#### Necesario tener .env con al menos

1. JWT_SECRET_KEY
2. DB_URL
